## How to Install Banner Shell In Termux.

`$ cd $HOME`

`$ pkg up -y`

`$ pkg install git -y`

`$ git clone https://github.com/noobzcoder/parrot

`$ cd parrotshell`

`$ chmod +x *`

`$ bash install.sh`

`$ Yes`
Now Restart Termux.

## How To Uninstall Parrot Shell From Termux.

`$ cd $HOME`

`$ cd parrotshell`

`$ bash uninstall.sh`
Now Restart Termux
